# jornadapascal

Este é um projeto genérico para propor exemplos e exercícios voltado para pessoas com conhecimentos em outras liguagens
Será usado o delphi e ffreepascal (com lazarus) para os exemplos e eventualmente outras linguagens como javascript para integrações.

# Objetivo específico

- Trabalhar simultâneamente várias competências pessoalmente ou através de exemplos/exercícios.
- Armazenar links e e código de exemplos e exercícios
